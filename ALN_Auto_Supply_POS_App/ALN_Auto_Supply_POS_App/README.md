
# ALN Auto Supply POS

Desktop cashier app for:
- product catalog with prices and stock
- receipt panel with subtotal, discount, total, cash, and change
- checkout flow with receipt numbers
- receipt history
- admin-protected product and settings management
- SQLite storage
- text receipts in the `receipts/` folder
- `transactions.csv` updated after every finished purchase

## Run
```bash
python main.py
```

## Default admin code
`1234`
